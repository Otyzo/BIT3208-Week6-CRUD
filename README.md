#BIT3208-Week 6:Database Integration and CRUD Operations
##About this Project
This is my week 6 practical work for BIT3208(Advanced Web Design and Development).
The task this week was to connect a web application to a database and perform CRUD operations.
I used my shopmart project to perform these operations.
##What i learned this week.
-How to connect php to a MySQL database
-How to create (Insert) new records into the database
-How to read(display) records from the database
-How to update existing records
-How to delete records
-How to validate users input before saving it
-How to protect the database using prepared statements(to prevent SQL injections)
##Technologies Used
-PHP
-MYSQL
-HTML
-phpMyAdmin
-XAMPP(local server)
##What the system can do
This is a simple Student/Product management system(using shopmart)that can:
-Add new records(Create)
-View all records in a table(Read)
-Edit/Update a record(Update)
